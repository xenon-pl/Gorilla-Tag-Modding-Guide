## Asset & Programming APIs {#apis}

Don’t reinvent the wheel! There are several APIs available to simplify your modding journey. For example, [GorillaLibrary](https://github.com/GorillaTagModdingHub/GorillaLibrary) provides both Asset and Programming APIs, making it easier to interact with the game and manage assets.

Whether you’re importing custom assets or extending gameplay mechanics, these APIs will save you time and help maintain compatibility with Gorilla Tag updates.
